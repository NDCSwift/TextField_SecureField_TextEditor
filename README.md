# ⌨️ TextField, SecureField & TextEditor
A SwiftUI reference project covering all three text input views — single-line, password, and multi-line.

---

## 🤔 What this is
This project demonstrates the differences and use cases for SwiftUI's three text input components: `TextField` for single-line input, `SecureField` for password entry with hidden characters, and `TextEditor` for multi-line free-form text. It covers binding, formatting, keyboard types, focus management, and styling for each.

## ✅ Why you'd use it
- **TextField vs SecureField** — Shows when and how to use each, including toggling password visibility with a show/hide button
- **TextEditor** — Demonstrates multi-line input with scrolling, character counts, and placeholder workarounds
- **Focus & keyboard** — Covers `@FocusState`, `submitLabel`, `onSubmit`, and dismissing the keyboard

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/TextField_SecureField_TextEditor.git
cd TextField_SecureField_TextEditor
```

### 2. Open in Xcode
Double-click `TextField_SecureField_TextEditor.xcodeproj`.

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team** — select your team.

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique reverse-domain ID.

## 🛠️ Notes
- No third-party dependencies required.
- If you see a code signing error, verify Team and Bundle ID are set.

## 📦 Requirements
- Xcode 15+
- iOS 16+
